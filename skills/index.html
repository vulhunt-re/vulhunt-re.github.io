VulHunt Agent Skills

VulHunt integrates with AI coding assistants through the Model Context Protocol (MCP), enabling LLM-driven vulnerability research directly from your editor or agent framework.

Ready-to-use agent skills

Browse our skills marketplace for guided workflows covering decompilation, dataflow analysis, pattern matching, and more.

LLM Integration

VulHunt exposes its analysis capabilities through an MCP server, allowing integration with any agent framework that supports the Model Context Protocol, including Claude Code, Claude Desktop, and OpenAI Codex.

The MCP server provides a set of tools that, combined with VulHunt's Lua query interface, give LLMs flexible access to the full analysis engine. Rather than exposing a rigid set of endpoints, VulHunt lets the model decide dynamically which queries to execute based on the current analysis context.

MCP ToolDescription
open_projectOpen a project from a given path, with optional platform attributes
query_projectExecute a Lua script against the currently open project and return the result as JSON
update_function_nameUpdate the name of a function in the currently open project
set_function_notesSet notes for a function in the currently open project
get_function_notesGet the notes for a function in the currently open project
load_signaturesLoad FLIRT signature databases for function identification
load_typesLoad a type library for function type information

For an in-depth walkthrough of LLM-driven vulnerability triaging and hunting, see Agentic Vulnerability Research with VulHunt.

MCP Configuration

Add VulHunt's MCP server to your AI assistant:

Claude Code:

claude mcp add vulhunt-ce -- vulhunt-ce mcp --stdio -d /path/to/bias-data

Codex CLI:

codex mcp add vulhunt-ce -- vulhunt-ce mcp --stdio -d /path/to/bias-data

For full configuration options, see the MCP documentation.

Skills Marketplace

To guide LLMs through common analysis tasks, VulHunt ships a set of Agent Skills: structured instructions that teach an agent how to use VulHunt's MCP tools for specific workflows. Skills are hosted at github.com/vulhunt-re/skills and distributed as a plugin marketplace that can be installed with the following commands:

claude plugin marketplace add vulhunt-re/skills
claude plugin install vulhunt

For skill.sh users:

npx skills add vulhunt-re/skills

Once installed, skills are invoked via slash commands (e.g., /vulhunt:decompiler). Each skill provides the LLM with the right Lua query templates, parameter formats, and analysis strategies for a given task, so the model can act autonomously without needing to learn the full API upfront.

This initial set of skills can be extended to handle more complex analyses or target specific vulnerability patterns.

Available Skills
