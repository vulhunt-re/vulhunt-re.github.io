VulHunt Enterprise customers have access to VulHunt Software Development Kit (SDK), which includes among other tools:

- VulHunt Jupyter Kernel
- VulHunt Shell

Using a dedicated client (rather than a plain text editor) speeds up rule development and testing.

The next sections cover the Jupyter kernel installation and configuration alongside with VulHunt Shell and Jupyter Lab integration.
